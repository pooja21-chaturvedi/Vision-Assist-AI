# Vision Assist AI 👁️🤖

An AI-powered web application designed to empower visually impaired users by providing voice-controlled assistance, real-time object recognition, document reading, and accessibility-focused tools.

---

## 🌟 About the Project

**Vision Assist AI** is an intelligent accessibility platform that helps visually impaired users interact with computers and their surroundings using Artificial Intelligence.

The application combines **Natural Language Processing (NLP)**, **Computer Vision**, **Speech Recognition**, **Text-to-Speech (TTS)**, and **Generative AI** to create an intuitive, voice-first experience that promotes independence and accessibility.

---

## 🎯 Our Mission

Our mission is to make technology accessible for everyone by building AI-powered tools that enable visually impaired users to perform everyday digital tasks with confidence and ease.

Vision Assist AI provides a hands-free, voice-controlled interface that allows users to navigate applications, understand their surroundings, read documents, identify objects, and manage daily activities independently.

---

## 🚀 Key Features

### 🎤 Voice-Controlled Navigation

* Navigate the entire application using natural voice commands.
* Hands-free interaction with all major features.
* Audio feedback for every action.

### 🌍 Multi-Language Support

Supports multiple Indian languages including:

* English
* Hindi
* Tamil
* Telugu
* Bengali

### 🕒 Live Time & Weather

* Displays the current time.
* Provides real-time weather updates.

### 📧 Email Assistant

* Reads the number of unread emails.
* Can read email content using text-to-speech.

### 📷 AI Scene & Object Detection

* Detects everyday objects using the device camera.
* Provides spoken descriptions of detected objects.

### 💵 Indian Currency Detection

* Recognizes Indian banknotes using the camera.
* Announces the detected currency denomination through voice.

### 🖼️ Image Text Reader (OCR)

* Extracts text from images.
* Reads the detected text aloud using speech synthesis.

### 📄 Document Reader

* Reads documents aloud using intelligent Text-to-Speech.
* Supports accessible document interaction.

### ✍️ Voice-Based Text Editor

* Create and edit text documents entirely through voice commands.
* Supports dictation and voice editing.

### 💻 System Integration

* Voice-based interaction with supported computer functions and applications.
